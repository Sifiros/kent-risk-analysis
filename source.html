Machine learning in risk assessment for online payment

The project

Proposed by : Martin CAUSSANEL (mtc26), Benjamin ROUX (bsar3), Alexis LAURENT (afgl2), Maxime BEL (mjab4)

Nowadays, online payments are taking a huge part of how people pay everyday, with online shops such as Amazon and Alibaba making huge profits every year. In this context, security is obviously one of the biggest parts of payment. However, security can be bothersome for users, making them turn away from good security measures, putting themselves at risk. Making sure the entire payment process is seamless for the user and include security is thus important.
We implemented something that works with the 3DS2 payment protocol that is coming out this year. This work would use machine learning to implement a risk assessment at the time of the payment, in order to determine if additional security checks are necessary, and if so, which ones.
